Un algorithme est une procédure permettant de résoudre un problème. Le débat sur l’IA pointe souvent, à tort, vers des algorithmes intelligents qui pourraient même « battre » l’intelligence humaine. Rien n’est plus éloigné de la vérité : les algorithmes sont extrêmement stupides[6][7][8] : ils font exactement ce qu’on leur demande (sinon, cela indiquerait une IA forte). Mais ils le font extrêmement rapidement, ce qui suggère l’illusion d’être « intelligent ». Les algorithmes sont efficaces pour traiter rapidement de grandes quantités de données, comme en témoigne la reconnaissance de formes par les robots (comme le diagnostic de maladies, les voitures autonomes, les ordinateurs d'échecs, etc.). L’inventeur d’un tel algorithme est peut-être intelligent, mais il est inévitablement limité par le fait qu’il ne peut garantir que son algorithme est une solution à son problème si le contexte du problème ne peut être clairement défini. Le cadre de référence d’un tel algorithme est limité (comme dans le cas des échecs par exemple). Dans la vie réelle, cependant, les gens doivent faire face à une multitude de contextes qui se chevauchent en partie dans un cadre de référence large dans lequel le problème ne peut pas être clairement défini. Le pouvoir de l’intelligence humaine réside dans le fait que les gens peuvent changer rapidement de contexte, par exemple lors d’une solution partielle à un problème. C’est également la base de changements de paradigme dans la résolution de problèmes.

Les humains et les animaux reconnaissent généralement les modèles de manière inconsciemment, à moins que le modèle ne s'écarte de ce à quoi nous nous attendons. Cette méthode inconsciente pointe vers l’exécution d’un algorithme dans le réseau neuronal. En fait, toutes les actions qu’une personne effectue automatiquement, sans y penser, sont réalisées par des algorithmes situés dans le système nerveux central. La conscience ne devient active que lorsque l’algorithme stagne à cause d’un événement inattendu.

Certains algorithmes peuvent apprendre, c’est-à-dire qu’ils peuvent ajouter de nouveaux modèles qu’ils ne connaissent pas encore à leur base de données. Il existe un risque qu’une telle nouvelle tendance renforce un préjugé. Libérer une base de données des biais est extrêmement difficile et nécessite beaucoup d’intelligence humaine. En conséquence, toutes sortes d’algorithmes échouent régulièrement.
